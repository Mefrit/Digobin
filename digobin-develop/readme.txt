0. Создание ролика
    Для создания ролика необходим сценарий и слайды (psd) с основными сценами и персонажами
    Необходимо создать проект в папке assets/projects. (Например lesson1)
    Проект состоит из двух основных файлов, сценария ('index.json') и описания сцены ('scene.json' ) (все предварительные результаты можно посмотреть в плеере).   

    Файл сцены представляет собой json файл с массивом обьектов. В файле могут быть ссылки на другие файлы.
    Обьект имеет свойства имя, положение, размер и тип, от типа зависит какой ресурс загружается, изображение или файл с анимацией dragon


    Файл сценария ('index.json'). Массив состояний обьектов сцены.
    В каждом состоянии описывается активный персонаж, набор реплик, переходы на другие сцены, состояние самих персонаей, аудиоспрайты для реплик
    Файл сценария можно изменить в редакторе истории.

    Аудиоспрайы. Для каждого персонажа можно сделать озвучку. 
        !! В сценарии добавить список персонажей
        !! В сценарии сделать привязку аудиофайла к персонажу
        !! В редакторе истории сделать выбор сценария и создания нового
        !! В плееере сделать выбор сценария для воспроизвдения            

    Подготавливаетя файл или несколько файлов (потом склеиваются в один) со всеми репликами персонажа   
    В Audacity убираются шумы, постронние звуки, взодхи длинные пацзы и т.п, для каждой реплики персонажа создается метка,
    между репликами делается промежуток примерно 1 сек. В html нет возможности остановить звук точно по времени.

    Файл со всеми репликами сохраняется в ogg, метки в txt. Названия файлов должны совпадать.

    В папке scripts, нужно запустить скрипт который разделит реплики на файлы, ("scripts/split.js") 
        потом для каждого файла распознаются фонемы ("scripts/speach2phonem.js") (используется sphinx, требует допиливания для лучшего распознования) потом они склеиваются в один файл с таймингом.
    В редакторе историй нужно назначить звуковые метки для каждой реплики. Или назначить автоматически по возможность ("scripts/setlabels.js")

    Структура файлов   
    Общие файлы (бибилотека, library)
        Для звуков, изображения, анимации dragon

    Файлы урока/проекта (projects)
        Для озвучки, для меток, для тайминга реплик
